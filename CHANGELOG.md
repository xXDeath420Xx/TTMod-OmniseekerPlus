# Changelog

All notable changes to OmniseekerPlus will be documented in this file.

## [1.0.0] - 2025-01-04

### Added
- Initial release
- Extended scanner range (up to 10x multiplier, max 2000m)
- New scannable types:
  - Hard Drives / Data Storage devices
  - Hidden chests and loot containers
  - Memory Trees (research core producers)
  - All ore vein types including Atlantum
- Six scan modes: All, Ores Only, Hard Drives, Loot, Memory Trees, Custom
- Quick scan hotkey (P) for instant area scan
- Scan mode cycling hotkey (O)
- In-game GUI (Shift+O) showing:
  - Last scan results sorted by distance
  - Mode selection buttons
  - Quick scan button
  - Settings toggles
- Distance-based color coding (green=close, yellow=medium, red=far)
- Configurable visual pulse effect on scan
- Full BepInEx configuration support
